Clockwise Installer

Instala el firmware en tu ESP32 directamente desde el navegador

Navegador no compatible
Este instalador requiere Google Chrome o Microsoft Edge en computadora de escritorio.

Requisitos

  • Navegador Chrome o Edge (en computadora)
  • Cable USB para conectar el ESP32
  • Drivers del ESP32 instalados (CP2102 o CH340)

Instalar Clockwise

Conecta tu ESP32 por USB y haz clic en el boton de abajo.
El proceso tomara aproximadamente 2 minutos.

Tu navegador no soporta Web Serial No se pudo acceder al puerto serial

Pasos de instalacion

1

Conecta el ESP32

Usa un cable USB de datos (no solo de carga) para conectar tu ESP32 a la computadora.

2

Haz clic en "Instalar Firmware"

Selecciona el puerto COM de tu ESP32 cuando aparezca la ventana.

3

Espera la instalacion

El proceso toma ~2 minutos. No desconectes el cable durante la instalacion.

4

Configura el WiFi

Al terminar, conectate a la red "Clockwise-WiFi" desde tu celular y sigue las instrucciones.
