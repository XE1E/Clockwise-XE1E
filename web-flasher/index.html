Instalar Firmware

Requisitos
  • Navegador con WebSerial (Chrome, Edge, Opera)
  • ESP32 conectado por USB
  • Cable USB con datos (no solo carga)
Tu navegador no soporta WebSerial. Usa Chrome, Edge u Opera en escritorio. WebSerial requiere HTTPS. Accede via https://xe1e.github.io/Clockwise-XE1E/web-flasher/

Instrucciones

Conecta tu ESP32 al puerto USB
Haz clic en "INSTALAR FIRMWARE"
Selecciona el puerto COM de tu ESP32
Espera a que termine el flasheo (~30 seg)
Configura WiFi con el asistente Improv
Accede a la IP del reloj para configurar
